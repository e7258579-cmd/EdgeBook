🔗 Google Sheets Sync

To sync your journal to Google Sheets, create an Apps Script Web App in your Google Sheet.

Setup Instructions:
  1. Open a new Google Sheet
  2. Extensions → Apps Script
  3. Delete existing code and paste the code from "Copy Script" below
  4. Save → Deploy → New deployment → Web app
  5. Execute as: Me  |  Who has access: Anyone
  6. Click Deploy, approve permissions, and copy the URL
